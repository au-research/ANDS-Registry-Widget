ANDS Registry Widget Demo
Look for a grant knowing the funder

Searching for a grant, filtering by the funders 

$('.scenario-1' ).registryWidget({
    apiKey: "public",
    mode: "search-activity",
    searchOptions: {
        autoSearch: true,
        facets: ['funders'],
        params: {
            type: 'grant',
            rows: 5,
            pp: 5
        }
    }
});

Advanced Search for a Grant

This configuration supports the default funder, rows and display of several facets 

<input type="text" id="search-grant" class="form-control" value=""/>
$('.advanced-search-grant').registryWidget({
    api_key: "public",
    mode: "search-grant",
    searchOptions: {
        autoSearch: true,
        facets: ['institutions', 'funders', 'type', 'fundingScheme'],
        params: {
            funder: 'Department of the Environment',
            rows: 5,
            pp: 5
        }
    }
});
